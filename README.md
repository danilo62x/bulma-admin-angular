# Bulma Admin / Angular

> Stack Angular 18 com signals

[![Licença: MIT](https://img.shields.io/badge/Licen%C3%A7a-MIT-blue.svg)](./LICENSE) ![Grátis](https://img.shields.io/badge/pre%C3%A7o-Gr%C3%A1tis-brightgreen)

Angular standalone components + signals + Bulma. Para times enterprise que precisam de um admin sólido com arquitetura previsível.

## 🧱 Stack

- Angular 18
- TypeScript
- Bulma
- RxJS

## ✨ Recursos

- Angular 18 standalone
- Signals e novo control flow
- Roteamento lazy
- Mesma identidade visual Bulma

## 🚀 Como rodar

Requer [Node.js](https://nodejs.org) 18+.

```bash
npm install
npm run dev
```

O app sobe em `http://localhost:4200`. Para gerar a versão de produção: `npm run build`.
